Это веб-приложение я написал для контроля своего бюджета и расходов. Идея была в том, что бы избавиться от google-sheets в пользу решения с простым вводом данных и с простым представлением данных.

Приложение писалось под требования конкретного заказчика (им был я сам), но в процессе разработки я решил добавить простую авторизацию, для тестирования этого приложения любым желающим.

По факту использования приложения появляются моменты из области UX, которые заносятся в файл todo.txt и по мере возможности дорабатываются. В целом опыт использования и отзывы заказчика — положительные. :)

Приложение реализовано для мобильных устройств, как SPA, на чистом Java Script. Для работы с базой данных написал небольшие php скрипты под каждый тип запроса.
